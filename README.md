### Descripción del proyecto
Este proyecto consiste en la creación de una aplicación web interactiva para visualizar datos de vehículos en venta. 

Tiene como objetivo mostrar la estructura de un proyecto de análisis de datos enfocado en una aplicación web, permitiendo a cualquier usuario explorar la información de manera visual y dinámica.

### Funcionalidades
La aplicación permite:

- Visualizar la distribución del kilometraje (odómetro) mediante histogramas
- Analizar la relación entre variables con gráficos de dispersión
- Interactuar con los datos a través de botones
- Explorar tendencias generales del mercado de vehículos

### Tecnologías utilizadas
- Python
- Pandas
- Plotly
- Streamlit

### Uso de la aplicación
El usuario puede acceder a la aplicación web desde el navegador y generar diferentes visualizaciones al hacer clic en los botones disponibles.

### Estructura del proyecto
- `requirements.txt`: versiones de las librerias que se utilizaron
- `EDA.ipynb`: contiene una breve exploracion de la base y las pruebas iniciales para definir las visualizaciones en la aplicacion web 
- `vehicles_project.py`: archivo principal de la aplicación
- `vehicles_us.csv`: conjunto de datos
- `README.md`: descripción del proyecto
